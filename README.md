# Next15 Project

A modern, scalable web application built with **Next.js 14** and the latest frontend technologies. This project is structured for performance, SEO optimization, developer experience, and rapid feature development.

---

## 🔐 Project Description

---

## 🧰 Technologies Used

- **Next.js 14** (App Router, Server Actions, Edge Rendering)
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **ESLint & Prettier** (Code formatting and linting)
- **ShadCN/UI** (Reusable components with Radix UI + Tailwind)
